# Roadmap

本路线图描述 Community Edition 的开源优先级。它不是承诺清单，而是帮助贡献者理解近期方向。

## v0.1 Community 起步版

- 本地项目创建、打开、保存。
- 无登录、无 VIP、无创作点限制。
- 基础素材管理、剧本编辑、场景编辑和预览。
- 默认手动音频，TTS 不绑定私有后端。
- README、LICENSE、NOTICE、CONTRIBUTING、SECURITY 和最小示例项目。

## v0.2 TTS Provider 完善

- 完善本地 HTTP TTS Provider 的 UI 配置。
- 增加 Provider 连接测试和错误提示。
- 补充 TTS Provider 文档和示例适配器。
- 优化无 TTS、手动音频和导出流程的提示。

## v0.3 示例项目和模板库

- 增加 1 到 3 个可再分发的 demo 项目。
- 增加可公开使用的基础背景、角色、道具或场景模板。
- 补充截图、短 GIF 或演示视频。
- 整理新手教程和常见问题。

## v0.4 导出稳定性

- 强化视频导出失败诊断。
- 优化长项目、长音频和多轨 BGM 导出。
- 补充导出链路测试。
- 增加浏览器兼容性说明。

## v0.5 插件和扩展

- 探索素材包、动作模板和 TTS Provider 的插件化边界。
- 梳理项目文件格式版本迁移策略。
- 评估桌面版或离线增强方案。
