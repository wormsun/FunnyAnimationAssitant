# AnimeStudio 图标设计说明

## 设计概念

AnimeStudio 的图标设计基于现有水印风格,采用现代化的渐变色彩和简洁的几何图形,体现了动漫短剧制作平台的专业性和创意性。

## 图标文件

### 1. logo.svg
- **尺寸**: 400×120px
- **用途**: 完整品牌 Logo，包含图标和文字
- **使用场景**: 
  - 网站头部
  - 文档封面
  - 宣传材料
  - 关于页面

### 2. favicon.svg
- **尺寸**: 128×128px
- **用途**: 浏览器标签页图标
- **使用场景**:
  - 浏览器标签页
  - 书签图标
  - PWA 应用图标

### 3. watermark.svg
- **尺寸**: 720×144px
- **用途**: 视频导出水印
- **使用场景**: 导出视频时的品牌标识

## 设计元素

### 色彩方案
- **主渐变**: 紫色 (#8B5CF6) → 粉色 (#EC4899) → 橙色 (#F59E0B)
- **文字颜色**: 深灰色 (#1F2937)
- **副文字**: 中灰色 (#6B7280)
- **强调色**: 红色 (#ff4444) - 用于录制指示灯

### 图标元素
1. **摄像机机身**: 代表视频创作
2. **镜头**: 象征专业性
3. **录制指示灯**: 表示实时创作
4. **播放按钮**: 体现播放和预览功能

### 字体
- **主字体**: Noto Sans SC (开源字体)
- **备选字体**: Microsoft YaHei, PingFang SC, Hiragino Sans GB
- **字重**: 
  - 主标题: 700 (Bold)
  - 副标题: 400 (Regular)

## 视觉效果

### 滤镜效果
1. **发光效果 (iconGlow)**: 为图标添加柔和的光晕
2. **阴影效果 (textShadow)**: 为文字增加深度感

### 设计原则
- **简洁**: 使用几何图形，避免过度装饰
- **现代**: 采用渐变色和扁平化设计
- **专业**: 保持视觉平衡和比例协调
- **可识别**: 在不同尺寸下都能清晰辨识

## 技术规格

### SVG 优势
- 矢量格式，无损缩放
- 文件体积小
- 支持动画和交互
- 浏览器原生支持

### 浏览器兼容性
- 所有现代浏览器均支持 SVG favicon
- 使用开源字体确保跨平台一致性

## 使用示例

### HTML 引用
```html
<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />

<!-- Logo 图片 -->
<img src="/logo.svg" alt="AnimeStudio" />
```

### CSS 背景
```css
.logo {
  background-image: url('/logo.svg');
  background-size: contain;
  background-repeat: no-repeat;
}
```

## 品牌一致性

所有图标设计都遵循以下原则，确保品牌视觉的一致性：
1. 使用相同的色彩方案
2. 保持相同的设计语言
3. 使用统一的字体系统
4. 维持一致的视觉风格
