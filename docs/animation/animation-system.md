## 1. 核心概念与定位

### 1.1 Animation 系统定位

```
┌──────────────────────────────────────────────────────────────────┐
│                      资源层 (Resource Layer)                      │
│  ┌─────────────┐  ┌───────────────┐  ┌─────────────────────────┐ │
│  │  Character  │  │     Prop      │  │      Background         │ │
│  │    角色     │  │     道具      │  │        背景             │ │
│  └──────┬──────┘  └───────┬───────┘  └────────────┬────────────┘ │
│         │                 │                       │              │
│         └─────────────────┼───────────────────────┘              │
│                           ▼                                      │
│              ┌──────────────────────────┐                        │
│              │  Animation Definitions   │  ← 资源级定义           │
│              │  (idle, speak, walk...)  │                        │
│              └──────────────────────────┘                        │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
                            │
                            ▼ 时间线触发
┌──────────────────────────────────────────────────────────────────┐
│                    动作层 (Action Layer)                         │
│              ┌──────────────────────────┐                        │
│              │   SetAnimationAction     │  ← 时间线触发点          │
│              │   { animName: "speak" }  │                        │
│              └──────────────────────────┘                        │
└──────────────────────────────────────────────────────────────────┘
                            │
                            ▼ 渲染层执行
┌──────────────────────────────────────────────────────────────────┐
│                    渲染层 (Render Layer)                         │
│              ┌──────────────────────────┐                        │
│              │   AnimationPlayer        │  ← 运行时播放器          │
│              │   (轨道解析 + 驱动执行)    │                        │
│              └──────────────────────────┘                        │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 核心概念定义

| 概念                | 定义                                         | 存储位置                          | 生命周期 |
| ------------------- | -------------------------------------------- | --------------------------------- | -------- |
| **Animation**       | 可复用的复合动画定义，由多个 Track 组成      | 资源内部 (`Character.animations`) | 持久化   |
| **Track**           | 单个属性维度的动画轨道（帧序列/变换/可见性） | Animation 内部                    | 持久化   |
| **Keyframe**        | 轨道中的关键帧数据（时间点 + 属性值）        | Track 内部                        | 持久化   |
| **AnimationPlayer** | 运行时播放器实例，负责解析和驱动动画         | 渲染层                            | 运行时   |
| **SceneAnimation**  | 🆕 场景级 Animation，绑定目标对象的动画效果   | `Scene.animations[]`              | 持久化   |

### 1.3 两种 Animation 使用方式

| 方式                 | 定义位置                       | 管理入口                   | 适用场景           |
| -------------------- | ------------------------------ | -------------------------- | ------------------ |
| **资源级 Animation** | Character/Prop/Background 资源 | 资源编辑器 → Animation Tab | 通用、可复用的动画 |
| **场景级 Animation** | `Scene.animations[]`           | 🆕 右侧侧边栏 → 动画 Tab    | 场景特定的动画效果 |

### 1.4 Animation vs Action 职责边界

> [!IMPORTANT]
> Animation 是临时的视觉效果，**不跨 block 累积**；Action 是持久的状态变更，**跨 block 累积**。

| 维度         | Animation        | Action                       |
| ------------ | ---------------- | ---------------------------- |
| **职责**     | 临时视觉效果     | 持久状态变更                 |
| **状态累积** | ❌ 不累积         | ✅ 跨 block 累积              |
| **典型场景** | 挥手、呼吸、晃动 | 位置、姿态、表情             |
| **控制层级** | 可通过 `targetObjectId` 控制对象自身或子对象 | 场景对象级状态变更 |

## 2. Animation 类型与支持范围

### 2.1 内置 Animation 类型（第一期）

针对用户最常见的需求，第一期支持以下语义化 Animation：

| Animation 名称 | 适用对象  | 描述         | 对应 Track 类型                                          |
| -------------- | --------- | ------------ | -------------------------------------------------------- |
| **`idle`**     | Character | 待机呼吸动画 | `TransformTrack (scale 微变)` 或 `EffectTrack (breathe)` |
| **`speak`**    | Character | 说话动画     | `FrameSequenceTrack` + `TransformTrack`                  |
| **`nod`**      | Character | 点头动画     | `TransformTrack (Y 位移)`                                |
| **`shake`**    | Character | 摇头动画     | `TransformTrack (X 往返)`                                |
| **`bounce`**   | Prop/BG   | 弹跳强调     | `TransformTrack (scale 脉冲)`                            |
| **`float`**    | Prop/BG   | 漂浮动画     | `TransformTrack (Y 正弦)` 或 `EffectTrack (float)`       |
| **`pulse`**    | Prop/BG   | 呼吸光效     | `VisibilityTrack (alpha 循环)`                           |
| **`spin`**     | Prop      | 旋转动画     | `TransformTrack (rotation 持续)`                         |

### 2.2 Animation 轨道类型

```typescript
export type AnimationTrackType =
  | 'frame_sequence'   // 帧序列：控制帧动画播放
  | 'transform'        // 变换：x/y/scale/rotation 关键帧动画
  | 'visibility'       // 可见性：alpha 关键帧动画
  | 'effect'           // 动态特效：引擎内置算法
```

### 2.3 动态特效类型 (Effect Track) - 第一期

| 特效类型    | 效果描述              | 实现方式          | 工时  |
| ----------- | --------------------- | ----------------- | ----- |
| **wave**    | 波浪飘动              | SimplePlane       | 1.5d  |
| **breathe** | 呼吸起伏 (scale 脉动) | 纯 JS + Transform | 0.5d  |
| **float**   | 漂浮悬浮 (Y 位移)     | 纯 JS + Transform | 0.25d |
| **glow**    | 发光轮廓              | @pixi/filter-glow | 0.5d  |
| **trail**   | 拖影残像              | 多 Sprite 管理    | 1d    |

**总工时**：约 3.75 天

### 2.4 Track 作用范围

| 轨道类型         | 作用目标            | 说明             |
| ---------------- | ------------------- | ---------------- |
| `frame_sequence` | 可选 `targetPartId` | 不填则作用于整体 |
| `transform`      | 可选 `targetPartId` | 不填则作用于整体 |
| `visibility`     | 可选 `targetPartId` | 不填则作用于整体 |
| `effect`         | 可选 `targetPartId` | 不填则作用于整体 |

### 2.5 冲突解决策略

当多个 Animation 同时播放，且针对**同一对象**的**同一属性**（如同时控制左臂旋转）时，采用 **覆盖 (Override)** 策略：

1.  **优先级规则**：后播放（或在列表中靠后）的 Animation 优先级更高。
2.  **覆盖行为**：高优先级的 Animation 的输出值将直接覆盖低优先级的输出值。
3.  **无混合**：不进行属性值的加法混合或加权平均。

## 3. 数据结构设计

### 3.1 核心类型定义 (`src/types/animation.ts`)

```typescript
// ===== Animation System Types (v11.0) =====

/**
 * 轨道类型枚举
 */
export type AnimationTrackType = 'frame_sequence' | 'transform' | 'visibility' | 'effect'

/**
 * 缓动函数类型
 */
export type EasingType =
  | 'linear'
  | 'easeIn' | 'easeOut' | 'easeInOut'
  | 'easeInQuad' | 'easeOutQuad' | 'easeInOutQuad'
  | 'easeInCubic' | 'easeOutCubic' | 'easeInOutCubic'
  | 'easeInSine' | 'easeOutSine' | 'easeInOutSine'
  | 'easeInElastic' | 'easeOutElastic' | 'easeInOutElastic'
  | 'easeInBounce' | 'easeOutBounce' | 'easeInOutBounce'

/**
 * 变换关键帧
 * 所有属性均为相对于基准的增量
 */
export interface TransformKeyframe {
  time: number              // 归一化时间 (0-1)
  x?: number                // X 偏移量 (像素)
  y?: number                // Y 偏移量 (像素)
  scaleX?: number           // X 缩放乘数 (1.0 = 无变化)
  scaleY?: number           // Y 缩放乘数
  rotation?: number         // 旋转角度 (弧度)
}

/**
 * 可见性关键帧
 */
export interface VisibilityKeyframe {
  time: number              // 归一化时间 (0-1)
  alpha?: number            // 透明度 (0-1)
}

/**
 * 帧序列轨道
 * 仅用于帧动画（PartAsset.type = 'animation'）
 */
export interface FrameSequenceTrack {
  trackType: 'frame_sequence'
  targetPartId?: string     // 可选：目标部件 ID，不填则作用于整体（道具/背景）
  assetId?: string          // 可选：指定帧动画素材 ID，不填则使用当前素材
  fps?: number              // 帧率，默认继承素材定义或 12
  loop?: boolean            // 是否循环
  // 时长由 frames.length / fps 自动计算
}

/**
 * 变换轨道
 * 使用 keyframes 定义关键帧（至少 2 帧）
 */
export interface TransformTrack {
  trackType: 'transform'
  targetPartId?: string     // 可选：目标部件 ID，不填则作用于整体
  duration?: number         // 动画时长 (ms)，默认 1000
  easing?: EasingType       // 缓动函数，默认 'linear'
  keyframes: TransformKeyframe[]  // 关键帧列表（至少 2 帧）
  
  // 旋转/缩放锚点（可选）
  pivot?: {
    x: number   // 相对于对象左上角的 X 偏移（像素）
    y: number   // 相对于对象左上角的 Y 偏移（像素）
  }
}

/**
 * 可见性轨道
 * 使用 keyframes 定义关键帧
 */
export interface VisibilityTrack {
  trackType: 'visibility'
  targetPartId?: string
  duration?: number         // 动画时长 (ms)，默认 1000
  easing?: EasingType
  keyframes: VisibilityKeyframe[]
}

export type DynamicEffectType = 'wave' | 'breathe' | 'float' | 'glow' | 'trail'

/**
 * 特效参数联合类型 (第一期 5 种)
 */
export type EffectParams =
  | { type: 'wave'; speed?: number; amplitude?: number; frequency?: number; direction?: 'horizontal' | 'vertical' }
  | { type: 'breathe'; intensity?: number; speed?: number }
  | { type: 'float'; amplitude?: number; speed?: number }
  | { type: 'glow'; color?: string; intensity?: number; size?: number }
  | { type: 'trail'; count?: number; fadeSpeed?: number; interval?: number }

/**
 * 特效轨道
 * 使用引擎内置算法，无需手动关键帧
 */
export interface EffectTrack {
  trackType: 'effect'
  targetPartId?: string     // 可选：目标部件 ID，不填则作用于整体
  effectParams: EffectParams
}

/**
 * 轨道联合类型
 */
export type AnimationTrack = FrameSequenceTrack | TransformTrack | VisibilityTrack | EffectTrack

/**
 * Animation 定义
 * 注意：不再有全局 duration，时长由各轨道自己决定
 */
export interface AnimationDefinition {
  id: string                // UUID
  name: string              // 语义化名称 (如 "speak", "idle")
  displayName?: string      // 显示名称 (如 "说话动画")
  description?: string      // 描述说明
  tags?: string[]           // 标签分类
  loop: boolean             // 是否循环，默认 false

  // 轨道列表（多个轨道可并行播放）
  tracks: AnimationTrack[]

  // 嵌套组合（可选，用于复合动画）
  children?: AnimationChildRef[]

  // 元数据
  createdAt?: number
  updatedAt?: number

  // 🆕 来源追踪（用于跨资源导入场景）
  sourceRef?: {
    resourceType: 'character' | 'prop' | 'background'
    resourceId: string           // 来源资源 ID
    resourceName?: string        // 来源资源名称（冗余存储，便于显示）
    animationId: string          // 来源 Animation ID
    animationName?: string       // 来源 Animation 名称
    copiedAt: number             // 复制时间戳
  }
}

/**
 * 子 Animation 引用
 */
export interface AnimationChildRef {
  animationId: string       // 引用的 Animation ID
  delay?: number            // 延迟开始 (ms)
}

/**
 * 🆕 场景级 Animation
 * 存储在 Scene.animations[]，通过右侧侧边栏管理
 */
export interface SceneAnimation {
  id: string                // UUID
  name: string              // 显示名称
  
  // 目标配置（可控制多个对象/部件）
  targets: {
    objectId: string        // 目标场景对象 ID
    partId?: string         // 可选：目标部件 ID，不填则作用于整体
  }[]
  
  // Animation 定义
  definition: AnimationDefinition
  
  // 播放控制
  autoPlay: boolean         // 是否自动播放，默认 true
  speed?: number            // 播放速度，默认 1.0
}
```

### 3.4 Action 层设计 (v11.0)

> [!NOTE]
> 重新设计 `set_anim` Action 结构，统一控制所有类型的动画。

```typescript
/**
 * 设置动画状态动作 (v11.0 重设计)
 * 统一控制所有类型的动画（帧序列、变换、特效等）
 */
export interface SetAnimAction extends BaseAction {
  type: 'set_anim'
  category: 'point'
  params: {
    /**
     * 动画名称
     * - 引用目标对象资源中定义的 Animation (如 "speak", "idle")
     * - 对于单纯的帧序列，系统自动生成隐式 Animation
     */
    animName: string
    
    /**
     * 控制指令
     */
    action: 'play' | 'stop' | 'pause' | 'resume'
    
    /**
     * 可选：播放参数
     */
    speed?: number        // 播放速度，默认 1.0
    loop?: boolean        // 覆盖 Animation 定义的循环设置
    reset?: boolean       // play 时是否从头开始，默认 true
  }
}
```

#### 3.4.1 隐式 Animation 机制

为让现有帧序列资产无缝工作，系统为每个帧动画部件自动生成**隐式 Animation**：

```typescript
// 示例：角色的 expression 部件有帧动画素材
// 系统自动生成：
{
  id: '__implicit_expression',
  name: '_expression',    // 以 _ 开头表示隐式
  duration: 1000,         // 根据 frames.length / fps 计算
  loop: true,
  tracks: [{
    trackType: 'frame_sequence',
    targetPartId: 'expression'
  }]
}
// 用户可通过 animName: '_expression' 控制
```

#### 3.4.2 使用示例

```typescript
// 播放说话动画
{ type: 'set_anim', target: 'char_01', params: { animName: 'speak', action: 'play' } }

// 以0.5倍速播放，不循环
{ type: 'set_anim', target: 'prop_star', params: { animName: 'spin', action: 'play', speed: 0.5, loop: false } }

// 停止发光特效
{ type: 'set_anim', target: 'char_01', params: { animName: 'magic_glow', action: 'stop' } }
```
