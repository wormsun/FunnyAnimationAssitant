# Changelog

All notable public changes to Funny Animation Assistant Community Edition are documented here.

## v0.1.0 - Initial Community Edition

### Added

- Local-first project creation, opening, and saving with `.anime` project files.
- Material management for characters, expressions, backgrounds, props, sound effects, and scene templates.
- Script editing, scene editing, preview, and basic video export flows.
- Optional local or self-hosted HTTP TTS provider configuration.
- Minimal demo project under `examples/demo-project`.
- Public README, roadmap, contribution guide, security policy, license, and notices.
- GitHub issue templates, pull request template, CI workflow, release checklist, and public release audit.

### Not Included

- Login, account system, VIP, payment, creation points, or recharge flows.
- Cloud project sync, private backend, admin console, or private deployment config.
- Bundled commercial material library or unclear redistribution assets.

### Known Limitations

- File System Access API support is best in Chrome and Edge.
- TTS provider UI and connection diagnostics are still early.
- Large project and long audio export stability need more validation.
